<h1 align="center">🎬 Movie Recommendation System</h1>
<h3 align="center">Collaborative Filtering & Similarity-Based Recommender</h3>

<hr/>

<h2>🚀 Project Overview</h2>
<p>
This project implements an intelligent movie recommendation system
designed to suggest relevant movies based on user preferences.
</p>

<p>
The system leverages collaborative filtering techniques and similarity-based methods
to generate personalized movie recommendations.
</p>

<hr/>

<h2>📊 Data Analysis</h2>
<p>
Exploratory Data Analysis was performed to understand movie distribution across genres.
</p>

<p align="center">
  <img src="Most Common Movie Genres.png" width="600">
</p>

<ul>
<li>Drama and Comedy dominate the dataset</li>
<li>Action and Thriller follow closely</li>
<li>Long-tail genres include Film-Noir and Western</li>
</ul>

<hr/>

<h2>🧠 Recommendation Techniques</h2>

<ul>
<li>Collaborative Filtering</li>
<li>User-based similarity</li>
<li>Item-based similarity</li>
<li>Matrix factorization (Surprise library)</li>
</ul>

<h2>⚙️ Interactive Interface</h2>
